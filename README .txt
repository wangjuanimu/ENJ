******************************************************************************

               ENJ : An Improved Version of the Improved NJ Algorithm

******************************************************************************

ENJ is an implementation in Java of the algorithm of Hongyan et al. 
ENJ takes as input a distance matrix. 
As output, the program provides the user with a unrooted phylogenetic tree in newick format. 


The program can be started by using the following command:

----------------------------------------------
./ENJ.jar --input infile --output outfile
----------------------------------------------
